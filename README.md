## 🚀 Запуск

### Вариант 1: Python (самый быстрый)
```bash
cd frontend
python -m http.server 8001
```
**Откройте**: http://localhost:8001

### Вариант 2: Docker
```bash
docker-compose up -d
```
**Backend**: http://localhost:8000  
**Frontend**: http://localhost:8001

### Вариант 3: Node.js
```bash
cd frontend
npx http-server -p 8001
```

### Вариант 4: VS Code Live Server
Кликните правой кнопкой на `frontend/index.html` → `Open with Live Server`

## 🗄️ Работа с базой данных и админ-панель
- По умолчанию данные хранятся в PostgreSQL контейнере `db`.
- Страницы `Товары`, `Корзина`, `Избранное`, `Заказы` открываются как отдельные разделы и автоматически подгружают актуальные данные при каждом переходе.
- Админ-панель доступна после входа под учётной записью:
  - Email: `admin@example.com`
  - Пароль: `Admin1234`
- Откройте фронтенд: `http://localhost:8001`
- После входа нажмите `Админ` в навигации.

### Как посмотреть содержимое базы данных в контейнере
```bash
# Войти в контейнер PostgreSQL
docker compose exec db psql -U tim -d ecommerce_db

# Просмотреть таблицы
\dt

# Примеры запросов
select * from users;
select * from products;
select * from cart_items;
select * from wishlist;
select * from orders;
select * from order_items;
```

### Если используется SQLite вместо PostgreSQL
Если в `.env` задано `DB_USE_SQLITE=true`, данные хранятся в файле:
```bash
backend/ecommerce.db
```
