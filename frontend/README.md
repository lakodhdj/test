# E-Commerce Frontend

Минималистичный фронтенд для e-commerce приложения.

## Функции

### 👤 Аккаунт
- Регистрация новых пользователей
- Вход в аккаунт с сохранением сессии
- Просмотр профиля
- Смена пароля
- Выход из аккаунта

### 📦 Товары
- Просмотр полного каталога товаров
- Поиск по названию
- Фильтрация по:
  - Категориям
  - Полу (мужской, женский, унисекс)
  - Размерам (XS, S, M, L, XL, XXL)
  - Цене (минимум и максимум)
- Пагинация (20 товаров на странице)
- Просмотр деталей товара
- Наличие/отсутствие товара

### 🛒 Корзина
- Добавление товаров в корзину
- Удаление товаров из корзины
- Изменение количества товаров
- Просмотр итоговой стоимости
- Хранение корзины в базе данных на сервере

### ❤️ Избранное
- Добавление/удаление товаров в избранное
- Просмотр избранных товаров
- Быстрое добавление в корзину из избранного
- Хранение избранного в базе данных на сервере

### 📋 Заказы
- Оформление заказа с адресом доставки
- Просмотр истории заказов
- Отслеживание статуса заказа
- Просмотр деталей каждого заказа

## Стиль

- **Минималистичный дизайн**: Чистый, простой и функциональный интерфейс
- **Адаптивный**: Работает на desktop, tablet и mobile
- **Быстрый**: Легкий JS без зависимостей
- **Удобный**: Интуитивный пользовательский интерфейс

## Запуск

### 1. Убедитесь, что бэкенд запущен

```bash
cd backend
python -m uvicorn app.main:app --reload
```

Сервер будет доступен на `http://localhost:8000`

### 2. Откройте фронтенд

Просто откройте `frontend/index.html` в браузере или используйте простой веб-сервер:

```bash
# Python 3
python -m http.server 8001 -d frontend

# или через Live Server в VS Code
```

Фронтенд будет доступен на `http://localhost:8001`

## API Интеграция

Фронтенд работает с следующими API endpoints:

### Account
- `POST /account/register` - Регистрация
- `POST /account/login` - Вход
- `GET /account/me` - Получить профиль
- `POST /account/change-password` - Изменить пароль

### Products
- `GET /product` - Список товаров с фильтрацией
- `GET /product/{slug}` - Деталь товара

### Categories
- `GET /product/categories/` - Список категорий

### Cart
- `POST /cart/{product_id}` - Добавить в корзину
- `DELETE /cart/{product_id}` - Удалить из корзины

### Wishlist
- `POST /wishlist/{product_id}` - Добавить в избранное
- `DELETE /wishlist/{product_id}` - Удалить из избранного

### Orders
- `POST /orders/` - Создать заказ
- `GET /orders/` - Список заказов пользователя

## Особенности

### Хранение данных
Корзина, избранное и заказы сохраняются на сервере в базе данных. Фронтенд не использует localStorage для этих сущностей.

### Cookies
Токены аутентификации хранятся в httpOnly cookies для безопасности.

### Админ-панель и управление базой
- Войдите под `admin@example.com / Admin1234`
- После входа нажмите `Админ` в навигации
- В продакшен-режиме данные корзины, избранного и заказов сохраняются в базе данных

### Messages
Система уведомлений показывает:
- Успешные операции (зеленый цвет)
- Ошибки (красный цвет)
- Информационные сообщения (синий цвет)

## Разработка

### Добавление новой функции

1. Добавить HTML разметку в `index.html`
2. Добавить CSS стили в `styles.css`
3. Добавить логику в `main.js` в объект `app`

### Поддерживаемые браузеры

- Chrome 60+
- Firefox 55+
- Safari 12+
- Edge 79+

## Производство

Для развертывания в продакшене:

1. Измените `API_URL` в `main.js` на URL вашего продакшн-сервера
2. Используйте HTTPS вместо HTTP
3. Разверните фронтенд на веб-сервер (Nginx, Apache, или облачный сервис)
